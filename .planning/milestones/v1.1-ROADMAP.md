## Overview

Enhanced reading experience with intelligent timing, font options, and visual refinements. Building on v1.0 MVP to create a more polished, comfortable reading experience.

## Phases

### Phase 5: Reading Timing

**Goal**: Intelligent word timing based on punctuation and word length
**Depends on**: Phase 4 (v1.0 complete)
**Plans**: 1 plan

Plans:
- [x] 05-01: Variable word timing with length/punctuation adjustments

**Details:**
- Implemented calculateWordDelay() for per-word timing
- 15% extra time per character above 6 chars
- 50% bonus for sentence-ending punctuation (.!?)
- 25% bonus for pause punctuation (,;:)
- setTimeout chaining replaces fixed setInterval

### Phase 6: Font System

**Goal**: Google Fonts integration with "f" key cycling through font options
**Depends on**: Phase 5
**Plans**: 1 plan

Fonts supported:
- Times New Roman (default, local)
- Tinos (Google Font)
- EB Garamond (Google Font)
- Merriweather (Google Font)
- Libre Baskerville (Google Font)
- Spectral (Google Font)

Plans:
- [x] 06-01: Bundle Google Fonts locally, implement font cycling with persistence

**Details:**
- 5 Google Fonts bundled as .woff2 (~142KB total)
- 'f' key cycles through fonts
- Font preference persists via localStorage
- Per-font scaling for visual size normalization

### Phase 7: Visual Polish

**Goal**: Smoother transitions and ORP baseline alignment fix
**Depends on**: Phase 6
**Plans**: 1 plan

Plans:
- [x] 07-01: Smooth word transitions, ORP baseline alignment fix

**Details:**

## Milestone Summary

**Key Decisions:**
- 15% length bonus + punctuation pauses for natural reading rhythm
- Bundle fonts locally (woff2) for offline support
- 50ms transitions fast enough for high WPM reading
- Wrapper pattern for ORP baseline alignment

- ORP baseline misalignment across different fonts
- Jarring word transitions at high speeds
- Last word remaining on screen at end of document
