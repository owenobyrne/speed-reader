# Milestone v1.0: MVP

**Status:** SHIPPED 2026-01-18
**Phases:** 1-4
**Total Plans:** 6

## Overview

Build a cross-platform RSVP reader from scratch: scaffold the Electron app, implement the core word-presentation engine with ORP highlighting, add file parsing for multiple formats, then polish with keyboard controls and persistence.

## Phases

### Phase 1: Foundation

**Goal**: Working Electron app with basic window, project structure, and build pipeline
**Depends on**: Nothing (first phase)
**Plans**: 1 plan

Plans:
- [x] 01-01: Electron project setup and basic window

**Details:**
- Electron 40 with secure defaults (nodeIntegration: false, contextIsolation: true)
- Basic project structure with main.js, renderer.js, preload.js
- npm scripts for dev and build

### Phase 2: Core Reader

**Goal**: RSVP word display with ORP highlighting, the core visual experience
**Depends on**: Phase 1
**Plans**: 1 plan

Plans:
- [x] 02-01: RSVP display engine with ORP highlighting and visual design

**Details:**
- ORP calculation at 40% position for comfortable reading
- Black background, white Times New Roman text, red ORP letter
- Guide lines with center tick marks
- Default 300 WPM playback

### Phase 3: File Handling

**Goal**: Accept files via drag-drop, extract text from PDF, Word docs, and web URLs
**Depends on**: Phase 2
**Plans**: 2 plans

Plans:
- [x] 03-01: File infrastructure and document parsing
- [x] 03-02: Web URL text extraction

**Details:**
- IPC architecture for secure file access
- pdf-parse v1.1.1 for PDF extraction
- mammoth.js for Word document extraction
- Readability algorithm for URL article extraction
- Lazy-loaded dependencies for fast startup

### Phase 4: Controls & Polish

**Goal**: Full keyboard control, speed adjustment, navigation, and position persistence
**Depends on**: Phase 3
**Plans**: 2 plans

Plans:
- [x] 04-01: Keyboard shortcuts (speed, pause/play, font size)
- [x] 04-02: Navigation (jump sentence/paragraph) and position saving

**Details:**
- Space for play/pause
- Up/Down arrows for speed (100-1000 WPM in 50 WPM steps)
- +/- for font size (24-96px in 8px steps)
- Left/Right arrows for sentence navigation
- Ctrl+arrows for paragraph navigation
- Quick-tap navigation (media player style)
- localStorage position persistence per document
- Progress bar and position indicator

---

## Milestone Summary

**Key Decisions:**

- Electron over Tauri (user preference, familiar ecosystem)
- Times New Roman font (classic readability)
- ORP at 40% position (research-based optimal recognition)
- IPC architecture for parsing (security with contextIsolation)
- Lazy-load heavy dependencies (fast startup)
- Quick-tap navigation threshold 500ms (media player UX)

**Issues Resolved:**

- None major - development proceeded smoothly

**Issues Deferred:**

- ORP baseline alignment with surrounding word parts (visual polish)

**Technical Debt Incurred:**

- None significant
